Mac Kısayolları

  • Baha Erdoğan
  • Teknoloji
  • 0

Windows'tan macOS'a geçiş zorlayıcı olabilir. Bu rehber, klavyeyi verimli kullanmanıza yardımcı olmak için hazırlandı. Bir hata görürseniz e‑posta ile haber verebilirsiniz.

Genel Kısayollar

  • ⌘ + C – Kopyala
  • ⌘ + X – Kes
  • ⌘ + V – Yapıştır
  • ⌘ + Z – Geri Al
  • ⌘ + Shift + Z – Yinele
  • ⌘ + A – Tümünü Seç
  • ⌘ + Q – Uygulamadan Çık
  • ⌘ + W – Pencereyi Kapat
  • ⌘ + Tab – Uygulama Değiştir

Finder ve Dosya Yönetimi

  • ⌘ + N – Yeni Finder Penceresi
  • ⌘ + Shift + N – Yeni Klasör
  • ⌘ + Delete – Çöpe Taşı
  • ⌘ + Shift + Delete – Çöpü Boşalt
  • ⌘ + Space – Spotlight Arama
  • ⌘ + Option + Esc – Uygulamaları Zorla Kapat

Ekran Görüntüsü ve Kayıt

  • ⌘ + Shift + 3 – Tüm Ekranı Yakala
  • ⌘ + Shift + 4 – Seçili Alanı Yakala
  • ⌘ + Shift + 5 – Ekran Görüntüsü ve Kayıt Seçenekleri
  • ⌘ + Shift + 6 – Touch Bar'ı Yakala (varsa)

Tarayıcı Kısayolları

  • ⌘ + T – Yeni Sekme Aç
  • ⌘ + W – Sekmeyi Kapat
  • ⌘ + Shift + T – Son Kapatılan Sekmeyi Aç
  • ⌘ + R – Sayfayı Yenile
  • ⌘ + L – Adres Çubuğunu Seç
  • ⌘ + D – Sayfayı Yer İmlerine Ekle

Metin Gezinme

  • Option + → / ← – İmleci Bir Kelime Taşı
  • ⌘ + → / ← – Satır Sonu/Başına Git
  • ⌘ + Up/Down – Belgenin Başına/Sonuna Git
  • ⌘ + Shift + → / ← – Satır Bazında Seç
  • Option + Shift + → / ← – Kelime Bazında Seç

Mission Control ve Pencere Yönetimi

  • Control + ↑ – Mission Control'ü Aç
  • Control + ↓ – Uygulama Pencerelerini Göster
  • ⌘ + H – Geçerli Uygulamayı Gizle
  • ⌘ + M – Pencereyi Küçült
  • ⌘ + Option + M – Tüm Pencereleri Küçült

Çeşitli

  • ⌘ + , (Virgül) – Uygulama Tercihlerini Aç
  • ⌘ + Option + D – Dock'u Göster/Gizle
  • Fn + Q – Hızlı Not (macOS Monterey ve sonrası)

Geliştiriciler ve İleri Kullanıcılar

Geliştirici ve Terminal Kısayolları

  • ⌘ + ` – Aynı Uygulamanın Pencereleri Arasında Geçiş
  • ⌘ + Shift + [ / ] – Uygulama Sekmeleri Arasında Geçiş
  • ⌘ + Option + C – Finder'da Dosya Yolunu Kopyala
  • ⌘ + Shift + . – Gizli Dosyaları Göster/Gizle
  • ⌘ + Shift + V – Biçimlendirmeden Yapıştır

Terminal Kısayolları

  • Control + A – Satır Başına Git
  • Control + E – Satır Sonuna Git
  • Control + U – İmleçten Başa Kadar Sil
  • Control + K – İmleçten Sona Kadar Sil
  • Control + L – Terminal Ekranını Temizle
  • Control + R – Komut Geçmişinde Ters Arama
  • ⌘ + K – Terminal Çıktısını Temizle

Kod Düzenleyici (VS Code, Xcode)

  • ⌘ + P – Hızlı Dosya Aç
  • ⌘ + Shift + F – Projede Genel Arama
  • ⌘ + Option + F – Bul ve Değiştir
  • ⌘ + Shift + ] / [ – Açık Sekmeler Arasında Geçiş
  • ⌘ + Shift + X – VS Code Uzantıları'nı Aç
  • ⌘ + / – Satırı Yorumla/Yorumu Kaldır
